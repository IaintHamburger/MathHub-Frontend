# MathHub 數學題庫練習網站
## 主旨在於讓各位高中生練習數學，取得自己理想的成績！
網站中包含各類數學競賽/大考題/常見題型，以及完整的詳解，目的在於讓大家練習各種解題技巧！   
如果想幫我們開發網站，或是發現任何bug，歡迎聯絡IaintHamburger，或提交issue/PR/MR來幫助我們變得更好！   
(網站仍處於開發階段中，如果有興趣的話歡迎聯繫我！）
